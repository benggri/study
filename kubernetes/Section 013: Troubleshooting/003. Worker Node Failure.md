# Worker Node Failure

```bash
kubectl get nodes
kubectl describe node <node_name>
```

* Node 에서 사용 가능한 CPU 메모리와 Disk 공간을 확인한다

```bash
top
df -h
```

* kubelet 상태를 확인한다

```bash
service kubelet status
sudo journalctl -u kubelet
```

* 증명서를 확인한다

```bash
openssl x509 -in /var/lib/kubelet/<file_name>.crt -text
```
