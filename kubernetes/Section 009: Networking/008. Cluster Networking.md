# Cluster Networking

## IP & FQDN

```
 --------------------    -------------------    ------------------- 
| 192.168.1.10       |  | 192.168.1.11      |  | 192.168.1.12      | => IP Address 
| master-01          |  | worker-01         |  | worker-02         | => Host Name 
| 02:42:75:98:18:4a  |  | 02:42:ac:11:11:72 |  | 02:42:ac:11:11:32 | => MAC Address 
|                    |  |                   |  |                   | 
| Master Node        |  | Worker Node       |  | Worker Node       | 
|       ------       |  |      ------       |  |      ------       | 
 ------| eth0 |------    -----| eth0 |------    -----| eth0 |------  => Network 에 연결된 Interface 
        ------                 ------                 ------ 
           |                      |                      | 
           |               -------------                 | 
            --------------| Network     |---------------- 
                          | 192.168.1.0 | 
                           ------------- 
```
