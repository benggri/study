# Kubernetes Software Versions

* kubernetes cluster 를 설치할 때 특정 version 을 설치한다

```bash
kubectl get nodes

NAME STATUS ROLES AGE VERSION
```

```
v    1     .11               .3
     Major  Minor             Patch
            - Features        - Bug Fixes
            - Functionalities 
```
